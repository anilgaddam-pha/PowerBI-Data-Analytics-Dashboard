# PowerBI-Data-Analytics-Dashboard
Recruitment &amp; Hiring Analytics Dataset containing company details, job roles, HR contacts, and location-based hiring information. Interactive Data Analytics Dashboard built using Power BI with KPI tracking, data modeling, and dynamic visualizations for business insights.

This project involves the development of an interactive Recruitment & Hiring Analytics Dashboard using Power BI to analyze company and job opportunity data. The dataset includes company information, job roles, hiring status, job locations, employment type, revenue details, and HR contacts. The project focuses on transforming raw recruitment data into meaningful insights through data cleaning, modeling, and visualization. The dashboard enables users to monitor hiring trends, analyze job distribution, and support data-driven recruitment decisions.
